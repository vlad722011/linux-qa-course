# 🔹 Глава 3. Потоки, пайпы и работа с текстом — QA Cheatsheet

---

## 3.1 stdin/stdout/stderr
~~~bash
echo "Hello" > output.txt
echo "error" 2> error.log
ls /nonexistent &> all.log
~~~

---

## 3.2 Пайпы (|)
~~~bash
cat /var/log/syslog 
| grep ssh 
| tail -n 20
~~~

---

## 3.3 Текст: кирпичики
~~~bash
head -n 20 file
tail -n 50 file
tail -f file
less file
grep -i "pattern" file
wc -l
sort
uniq -c
~~~

---

## 3.4 QA-паттерны логов
~~~bash
sudo grep "Failed" /var/log/auth.log 
| tail -n 50

sudo grep "Failed password" /var/log/auth.log 
| awk '{print $(NF-3)}' 
| sort 
| uniq -c 
| sort -nr 
| head
~~~

---
