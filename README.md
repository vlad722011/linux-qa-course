# Linux QA Course (Ubuntu 24.04 VPS)

Личный учебник: Linux как инструмент диагностики и эксплуатации тестовой среды для QA (ориентация на mid+).
Практика выполняется на удалённом сервере Ubuntu 24.04 через SSH.

## Структура репозитория
- `cheatsheet/` — короткие шпаргалки (быстро вспомнить команды).
- `guide/` — подробные уроки (идея → практика → задачи → мини-инцидент).

## Содержание (актуальные файлы)
### Cheatsheet
- `cheatsheet/01-shell-streams.md` — терминал, shell, stdin/stdout/stderr, редиректы.
- `cheatsheet/02-pipes-redirects.md` — пайпы и редиректы, быстрые цепочки для логов.

### Guide
- `guide/01-shell-streams.md` — подробно про shell и потоки + практика и задания.
- `guide/03-pipes.md` — пайпы и обработка текста для логов.
- `guide/04-journalctl-ssh.md` — journalctl и SSH-диагностика под Ubuntu 24.04.

## Правило обучения
1) Прочитал идею (guide).
2) Повторил руками на VPS.
3) Сдал задачи (файлы/вывод).
4) Разобрал мини-инцидент (как на работе).
