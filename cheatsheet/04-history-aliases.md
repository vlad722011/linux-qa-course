# Cheatsheet 04: History + Aliases — QA

Цель: экономить время на повторяющихся командах, понимать разницу alias/function/binary.

---

## 1) History Expansion (быстро!)
~~~bash
**!!**              # повторить **последнюю** команду
**!\***             # повторить **все аргументы**
**!\$**             # **последний аргумент**
**!#**              # выполнить **уже набранное**
**Ctrl+R**          # **поиск** по истории
~~~

**Пример:**
~~~bash
cat /etc/passwd
head -5 **!\$**        # → head -5 /etc/passwd
~~~

---

## 2) Диагностика команд
~~~bash
**type -a** ls        # alias → builtin → binary
**command -V** ls     # тип команды  
**which** ls          # путь к бинарнику
~~~

---

## 3) Jobs (фон)
~~~bash
sleep 60 **&**        # запуск в фон
**jobs**              # список задач
**kill %1**           # убить по номеру job
**fg %1**             # вернуть на передний план
~~~

**QA-tip:** `%sleep` — kill по имени job.
